# mimamori

見守り機能基板のためのプログラムです。

人感センサーにより人を感知して、
メール、LINE、twitterで通知するとともに
ambientへのデータを投げてのダッシュボード表示などを行います。

これらは、基本無料サービスの範囲で使用できるものであり、
見守り機能を無料で実現できます。
ただし、通信についてはwifiか通信サービスを利用する必要があります。

また、これらサービスを無料で使うためにそれぞれのサービスにご自身で
登録する必要があります。


git clone https://github.com/momorara/mimamori
でラズパイにダウンロードしてください。
